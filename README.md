# python_edu_mgmt
education management

This app is mangeing universites, schooles aand students.

This app contains DELETE, POST, GET, PUT function based apis. Every api has meta data information and it's URLs.
For login session used csrf_exempt in apis. All apis are created with serializers.

Admin panel contains ordering, filtering and serching.

Model contains custom emthod to get custom fields.

Serializers contains custom methods, method fields and also use custom create method in serializer.
